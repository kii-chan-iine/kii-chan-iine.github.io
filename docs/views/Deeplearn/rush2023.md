# AutoML

## 简单

1. 网格:  n2
2. 随机:  n

## 贝叶斯

1. BO-GP:  n3
2. BO-SMAC:   nlogn
3. BO-TPE:  nlogn

## 启发式

1. GA:n2
2. PSO:nlogn

## 高保真

1. SH
2. HB:  nlogn
3. BOHB： nlogn

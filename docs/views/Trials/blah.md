出差，跟某个学会的Fellow学习，感受复杂



学到的东西：

1. 一定要新，写摘要，一定要突出自己的卖点，1-2条就够了，不要太多
2. 很多人共同的创作一定要组织好，合作要谈好，否则很多问题。也会导致作品出的很慢。
3. Fellow语言能力是真的强且自信，这个需要自己加强
4. Results的部分，先把自己的结果呈现出来，最后的部分只有少部分文献的引用，说现有的研究支撑了我们的结果。否则‘你是你是发表别人的成果么？这又什么用呢?’

问题：

1. 合作搞清楚，否则后面问题多
2. 团队要职责明确，一直依靠大佬可能会拖进度
3. 犹犹豫豫也会拖进度

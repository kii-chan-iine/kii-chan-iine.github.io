# 距离

设X是一非空集合，任给一对这集合中的元素x,y都给定一个实数d(x,y)与他们对应，并且满足：

<img src="http://latex.codecogs.com/svg.latex?(1) \quad d(x,y) \ge0,d(x,y)=0 \Leftrightarrow  x=y" border="0"/>

<img src="http://latex.codecogs.com/svg.latex?(2) \quad d(x,y) =d(y,x)" border="0"/>

<img src="http://latex.codecogs.com/svg.latex?(3) \quad d(x,y) \le d(x,z)+d(z,y)" border="0"/>

则称d(x,y)为两点之间的**距离**。

# 范数（比距离多一个约束2）

设<img src="http://latex.codecogs.com/svg.latex?\left \| x \right \|" border="0"/>是<img src="http://latex.codecogs.com/svg.latex?R^n" border="0"/>的范数，则满足：

1. <img src="http://latex.codecogs.com/svg.latex?\left \| x \right \| \ge 0, \forall x \in R^n:\left\|x \right \|=0 \Leftrightarrow x=0" border="0"/>
2. <img src="http://latex.codecogs.com/svg.latex?\left \| \alpha x \right \|=|\alpha|\left \| x \right \|, \forall \alpha \in R, x \in R^n" border="0"/>
3. <img src="http://latex.codecogs.com/svg.latex?\left \| x+y \right \|=\left \| x \right \|+\left \|y\right\|, \forall x,y \in R^n" border="0"/>

即，比到原点的距离多了约束2。

# 内积空间

与我们常用的空间类似，角度、距离、长度等概念均存在。

内积定义：设<img src="http://latex.codecogs.com/svg.latex?(x,y)\in R" border="0"/>,且满足：

1. 对称性

2. 对第一变元的线性性（即，对于第一个变元，如果存在一个$\alpha$,这个$\alpha$可以被提出来）

3. 正定性

   则称<img src="http://latex.codecogs.com/svg.latex?(x,y)" border="0"/>为<img src="http://latex.codecogs.com/svg.latex?R" border="0"/>中的内积，而称<img src="http://latex.codecogs.com/svg.latex?R" border="0"/>为(实或复)内积空间，或准希尔伯特空间.

# 其他几个概念

1. 1904年Hilbert引入了无穷维的概念。无穷维的内积空间+完备性（指这个空间在极限运算中，取极限不能跑出去）就是Hilbert空间。
2. 1922年巴拿赫提出了赋范空间，其完备的赋范空间即巴拿赫空间。

# 鲁棒控制

## Hardy Spaces H 2 and H∞

### H2空间--H代指Hardy

<img src="http://latex.codecogs.com/svg.latex?H_2" border="0"/>是<img src="http://latex.codecogs.com/svg.latex?L_2(j\Re)" border="0"/>的一个（闭）子集，其矩阵函数<img src="http://latex.codecogs.com/svg.latex?F(s)" border="0"/>在<img src="http://latex.codecogs.com/svg.latex?Re(s)>0" border="0"/>（右开半平面）解析。相应的范数定义为：



<center>
<img src="http://latex.codecogs.com/svg.latex?\left \| F\right \|^2_2:=_{ \sigma >0 }^{ sup }{ \{ \frac { 1 }{ 2\pi  } \int _{ -\infty  }^{ \infty  } Trace[F*(\sigma +j\omega )F(\sigma +j\omega )]d\omega \}  }" border="0"/>
</center>

可以证明，

<center>
<img src="http://latex.codecogs.com/svg.latex?\left \| F\right \|^2_2= \frac { 1 }{ 2\pi  } \int _{ -\infty  }^{ \infty  } Trace[F*(j\omega )F(j\omega )]d\omega " border="0"/>
</center>



因此，可以像计算L2范数一样来计算H2范数。

## H2和H无穷

两个都是稳定且正定的系统（传递函数）的范数度量。他们定义在不同的信号空间，**无穷范数**代表了系统对峰值有界信号的传递特性，而**H2范数**则代表了系统对能量有界信号的传递特性。**无穷范数**问题常称为持续有界干扰抑制问题；**H2**则和LQG问题密切相关，是系统在白噪声激励下的稳态输出方差。两者都是代数线性控制中的方法。

## 鲁棒控制和H无穷控制

首先要明白什么是鲁棒，什么是H-infinity。这两个概念是独立的。；

（0）鲁棒

鲁棒指系统在不确定参数（parameter uncertainties）摄动下，系统维持其自身某种性能（稳定、某指标）的特性。鲁棒性的主要问题是高频的未建模动态。应该注意的是，我们在进行鲁棒稳定性分析时，要将干扰与控制输入置0。



（1）<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>
<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>是对传递函数增益大小的一个度量指标，简单说就是一个系统输入输出的放大倍数。<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>的标准问题实际是：求解已真有理（解析有界）的控制器K，使外输入W到加权输入Z的传递函数阵的H无穷范数最小。翻译过来就是，由于我们一般考虑线性系统，直接运用闭环系统稳定性分析的LMI进一步公式推导，直至导出矩阵元素中含有某一参数（性能指标）的LMI，运用MATLAB 求解即可。



(2)鲁棒<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>控制
鲁棒<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>控制就是抑制噪声到期望输出之间的传递函数集的最大增益，从而达到抗扰的目的。其实顾名思义，鲁棒H无穷就是研究系统既包含不确定性，又要求达到某一性能指标。鲁棒<img src="http://latex.codecogs.com/svg.latex?H_{\infty}" border="0"/>控制具有较强的保守性。



总结：研究系统鲁棒性，就是研究包含不确定参数的系统。研究H无穷，就是研究满足某性能指标的系统。

## 零状态，零输入和全响应（电路原理也有部分）

传递函数（网络函数）的定义就是零状态响应的拉普拉斯变换Y(s)与激励的拉普拉斯变换X(s)之比。即

<img src="http://latex.codecogs.com/svg.latex?H(s)=\frac{Y(s)}{X(s)}" border="0"/>

> 因为如果你想知道电路的全响应，可以根据叠加定理把全响应拆为[零状态响应](https://www.baidu.com/s?wd=零状态响应&tn=SE_PcZhidaonwhc_ngpagmjz&rsv_dl=gh_pc_zhidao)和储能器件激励下响应之和。而储能器件初始储能的数值是没办法知道的，所以不适合用全响应结果与激励的比值作为研究对象。零状态相应则不同，一旦电路结构确定下来，[零状态响应](https://www.baidu.com/s?wd=零状态响应&tn=SE_PcZhidaonwhc_ngpagmjz&rsv_dl=gh_pc_zhidao)就确定下来了，所以这样计算出来的传递函数是只与电路结构相关的结果。

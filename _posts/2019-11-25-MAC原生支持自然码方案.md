# macOS 原生输入法设置自然码

macOS 其实内置隐藏了双拼的自然码方案。

只需在终端内输入以下命令即可：

```c
$ defaults write com.apple.inputmethod.CoreChineseEngineFramework shuangpinLayout 5
```




> 此时即可使用自然码方案了。
>
> 附：系统为 macOS 10.14.4
